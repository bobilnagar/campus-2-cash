# Campus 2 Cash

A web app that helps students turn their projects into something that can actually earn money.

## What it does

You enter your project details, and the app gives:

* A score showing how useful your project is in the real world
* Possible ways to make money from it
* Basic pricing ideas
* Target users

## Built with

* Next.js
* React
* Tailwind CSS
* OpenRouter API

## How to run

```bash
git clone https://github.com/your-username/campus-2-cash.git
cd campus-2-cash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment

Create a `.env.local` file:

```
OPENROUTER_API_KEY=your_key_here
```

## Notes

* No database is used (data is stored locally)
* Built as a student project
